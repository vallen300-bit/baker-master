## S1 — test for `score_row` `unknown_non_canonical` guard

**File:** `tests/test_score_row.py` (new, 121 lines)

### Design

- Imports `score_row` from `scripts/run_kbl_eval.py` via `sys.path.insert(0, scripts/)`. Same pattern as the patched scripts themselves.
- Uses the existing `tests/fixtures/vault/` (3 slugs: alpha/beta/gamma) so tests are decoupled from the production 19-slug list. Same discipline as `test_slug_registry.py`.
- `monkeypatch.setenv("BAKER_VAULT_PATH", fixture)` + `slug_registry.reload()` per test, autouse — mirrors `test_slug_registry.py::_reset_cache`.
- Thin `_run(**kwargs)` wrapper builds label + parsed dicts with sane defaults so each test sets only the fields under examination.
