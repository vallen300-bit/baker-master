01The three teams

Each team has a distinct mode, distinct output, distinct rhythm.
Lab
Design Pass

Sketches new capabilities before code is written. Output is a ratified brief. Hours-pace. Higher unknowns.

Members: Research Agent (lead) · Director (ratifier)
Build
Ship Code

Cortex-3T M1+. Brief → dispatch → implement → review → merge → deploy. Days-pace. Runs autonomously per AI Head charter.

Members: AI Head A (lead) · AI Head B (reviewer) · B1–B4 (build) · Research Agent (briefs)
Business
Real Brisen Use

Director uses Baker on actual matters (Hagenauer, Cupial, MO VIE, AO, MRCI/Lilienmat). Live-pace. Failures are free data.

Members: Director (lead) · AO PM · MOVIE AM · AI Dennis · Research Agent (observation)
