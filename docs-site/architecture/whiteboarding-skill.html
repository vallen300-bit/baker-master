Steps 1–9 — intake to plan

#PhaseStepDescriptionDoerOutput
1Phase 0 · IntakeFrame the issue You indicate it; agent offers 2–3 candidate banners; you pick one. It then locks verbatim. You + agentIssue banner

3Phase 1 · ContextSweep all sources Helper agents sweep all 15 master-list surfaces — incl. Dropbox, Fireflies, Plaud. Every row logged. AgentCoverage log
4Allocate desk Covering desk owns by default. No desk? Agent proposes one — never a bare question. New desk = your order. AgentNamed owner

Safety rules

No skipped gates

  • No output before its Triaga
  • Plan live only after gate 4

No silent agents

  • New desk only on your order
  • No desk = question to you

No auto-send outside

  • Externals: you press Send
  • Internal auto-send only if authorized
TO START A RUN
Say "/whiteboarding". The agent asks its first question.
