# MOVIE KPI Framework

## Core Metrics

| Metric | Definition | Target | Benchmark Source |
|--------|-----------|--------|-----------------|
| Occupancy | Rooms sold / rooms available (~153 rooms) | [TO FILL — debrief] | STR Vienna luxury set |
| ADR | Average Daily Rate (EUR) | [TO FILL — debrief] | STR Vienna luxury set |
| RevPAR | Revenue Per Available Room | [TO FILL — debrief] | STR Vienna luxury set |
| GOP | Gross Operating Profit | [TO FILL — debrief] | MO monthly P&L |
| AGOP | Adjusted GOP = GOP minus basic fees minus royalty | [TO FILL — debrief] | Calculated |
| NOI | Net Operating Income = AGOP minus taxes/insurance (capped 2% GR) minus FF&E | [TO FILL — debrief] | Calculated |
| ANOI | Adjusted NOI = NOI minus Owner's Priority Return | Must be positive for incentive fees | Calculated |
| Owner's Priority Return | EUR 4.0M+ (Y5+, CPI-escalated from Y6) [MA 1.1] | See agreements_framework.md | Annual |
| FF&E Reserve | 4% of Gross Revenue (Y4+) [CSA 5.5] | Cumulative balance | Monthly deposits |

## Performance Test (MA 11.5) — starts FY 9

| Parameter | Value |
|-----------|-------|
| Test period | 2 consecutive Fiscal Years |
| RevPAR Test | Hotel RevPAR >= 3rd highest in Competitive Set |
| Threshold Test | AGOP >= 90% of budgeted AGOP OR positive ANOI |
| Failure = | Both tests fail in each of 2 consecutive FYs |
| Competitive Set | Park Hyatt Vienna, Hotel Imperial, Hotel Bristol, Sacher Hotel |
| Data source | Smith Travel Research (STAR) |
| Comp set review | Every 3 years from Opening Date |
| Cure | Mandarin may pay Cure Amount (max 2x during Initial Period) |

## Budget Variance Tolerances (CSA 3.4.6-7)
- **Per Cost Centre:** up to 7.5% overrun without Owner approval
- **Aggregate:** up to 5% of total budgeted expenditure without Owner approval
- Same limits apply to FF&E Budget
